# Anom Artsy Rebuild

This repo is the clean rebuild for Anom Artsy / Anom Originals.

## Goal
Build a site that feels like Anom:
- dark
- neon
- curated
- expressive
- feminine edge
- art-first, not bot-first

## Core sections
- Home
- Gallery
- Custom Work
- Merch
- About
- Contact

## Rules for this rebuild
- No giant product dump
- No generic filler text
- No mixed gallery/shop confusion
- No off-brand layout choices
- Keep the current live repo untouched until this version is ready

## Build order
1. Homepage
2. Gallery
3. Custom Work
4. Merch
5. About / Contact

## Notes
The old `anom-originals-landing` repo stays as the current live backup until this rebuild is finished.
